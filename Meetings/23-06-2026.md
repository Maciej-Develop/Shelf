# 1st Meeting 23/06/2026

## Project Idea :

Shelf will be a full-stack web application that allows users to explore and read reviews of various media, including books, movies, TV shows, manga, anime, and more.

## Technologies :

Frontend : Angular v22 (to verify)

Backend : Spring boot 4.1.0 (Java 21)

DB : PostgreSQL or MySQL

## Fonctionnalities : 

- A home page where users can browse different reviews
- A search function
- An admin/reviewer interface for adding reviews
- Users can create accounts
- Users can post comments on reviews (if we disagree with them, we nuke them)
- A voting system similar to YouTube's like/dislike feature or Reddit's upvote/downvote system
- A feature that displays a random review to help users discover new content
- On review pages, admins/reviewers can add recommendations they consider relevant
- A responsive design that allows users to navigate the site easily across different screen sizes
